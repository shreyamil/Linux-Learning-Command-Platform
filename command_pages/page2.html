Working With Directories

Directory Listing Commands

The ls command is used to list directory contents. Here are its main variations:

Command Description
ls Basic listing of current directory contents
ls path List contents of specified directory
ls -l Long listing format with details
ls -a Include hidden files (starting with .)
ls --author Show file owners
ls -S Sort by file size (largest first)
ls *.html, ls *.txt Filter by file extension
ls -lS > file.txt Save listing to a file

ls - List Directory Contents

  1. What it does: Lists all visible contents in the current directory
  2. When to use: To quickly see what's in your current location
  3. Example: 
    ls
Documents  Downloads  Pictures  Music  Videos

ls path - List Specific Directory

  1. What it does: Shows contents of another directory without navigating there
  2. When to use: When you want to peek into another directory
  3. Example: 
    ls /home/user/Documents
report.pdf  notes.txt  projects/

ls -l - Long Listing Format

  1. What it does: Shows permissions, owner, size, and modification time
  2. When to use: When you need detailed file information
  3. Example: 
    ls -l
total 4
drwxr-xr-x 2 user user 4096 Jan 10 09:15 Documents
drwxr-xr-x 2 user user 4096 Jan 10 09:15 Downloads
-rw-r--r-- 1 user user  102 Jan 10 09:15 notes.txt

ls -a - Show Hidden Files

  1. What it does: Includes files starting with . (hidden files)
  2. When to use: To view configuration files and other hidden items
  3. Example: 
    ls -a
.  ..  .bashrc  .profile  Documents  Downloads

ls --author - Show File Authors

  1. What it does: Displays the owner of each file
  2. When to use: To check file ownership
  3. Example: 
    ls --author
Documents  user
Downloads  user
notes.txt  user

ls -S - Sort by Size

  1. What it does: Orders files by size (largest first)
  2. When to use: To identify large files
  3. Example: 
    ls -S
largefile.txt (2.4MB)  mediumfile.txt (1.2MB)  smallfile.txt (256KB)

ls *.html and ls *.txt - Filter by File Type

  1. What it does: Lists only files matching the pattern
  2. When to use: To work with specific file types
  3. Examples: 
    ls *.html
index.html  contact.html  about.html  style.css

ls *.txt
notes.txt  todo.txt  ideas.txt  README.txt

ls > file.txt - Output to File

  1. What it does: Saves directory listing to a file
  2. When to use: To create a record of directory contents
  3. Example: 
    ls -lS > listing.txt
# Creates listing.txt with the command output
