Working With Directories: cd

Directory Navigation Commands

The cd (change directory) command is used to navigate between directories:

Command Description
cd or cd ~ Changes to the home directory
cd / Changes to the root directory
cd .. Moves to the parent directory
cd 'xx yy' Changes to a directory with spaces in its name

cd or cd ~ - Home Directory

  1. What it does: Changes to your home directory
  2. When to use: When you want to quickly return to your home folder
  3. Example: 
    cd
# or
cd ~
pwd
/home/username
Current directory: /home/username/projects/website
📁 projects
📁 website
📄 index.html
📄 style.css

cd / - Root Directory

  1. What it does: Changes to the root directory (top of filesystem)
  2. When to use: When you need to access system-wide files
  3. Example: 
    cd /
pwd
/
Current directory: /home/username
📁 home
📁 username
📁 Documents
📁 Downloads

cd .. - Parent Directory

  1. What it does: Moves up one directory level
  2. When to use: When you want to go back to the containing folder
  3. Example: 
    pwd
/home/username/projects/website
cd ..
pwd
/home/username/projects
Current directory: /home/username/projects/website
📁 projects
📁 website
📄 index.html
📄 style.css

cd 'xx yy' - Directory with Spaces

  1. What it does: Changes to a directory with spaces in its name
  2. When to use: When working with folders containing spaces
  3. Example: 
    cd 'my documents'
pwd
/home/username/my documents
Current directory: /home/username
📁 Documents
📁 'my documents'
📄 report.docx
📄 notes.txt
📁 Downloads
