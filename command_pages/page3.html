Working With Directories: cd

Directory Navigation Commands

The cd (change directory) command is used to navigate between directories:

Command Description
cd or cd ~ Changes to the home directory
cd / Changes to the root directory
cd .. Moves to the parent directory
cd 'xx yy' Changes to a directory with spaces in its name

cd or cd ~ - Home Directory

  1. What it does: Changes to your home directory
  2. When to use: When you want to quickly return to your home folder
  3. Example: 
    cd

cd .. - Parent Directory

  2. When to use: When you want to go back to the containing folder
