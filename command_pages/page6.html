Working With Files: sort

Sorting Commands

The sort command organizes file contents alphabetically or numerically:

Command Description
sort file Basic alphabetical sort
sort file1 file2 Sort contents of multiple files
sort -r file Reverse sort order
sort -f file Case-insensitive sort
sort -n file Numerical sort

sort file - Basic Sort

  1. What it does: Sorts lines alphabetically
  2. When to use: When you need ordered output
  3. Example: 
    sort fruits.txt
apple
banana
orange
Output will appear here
Original File: fruits.txt

sort file1 file2 - Multiple Files

  1. What it does: Combines and sorts multiple files
  2. When to use: When merging sorted data from sources
  3. Example: 
    sort file1.txt file2.txt
apple
banana
grape
orange
Output will appear here
Files to Sort:
File 1: fruits.txt
File 2: more_fruits.txt

sort -r file - Reverse Sort

  1. What it does: Sorts in reverse order (Z-A, 9-0)
  2. When to use: When you need descending order
  3. Example: 
    sort -r fruits.txt
orange
banana
apple
Output will appear here
Original File: fruits.txt

sort -f file - Case Insensitive

  1. What it does: Ignores case when sorting
  2. When to use: With mixed-case data
  3. Example: 
    sort -f mixed_case.txt
apple
Banana
cherry
Orange
Output will appear here
Original File: mixed_case.txt

sort -n file - Numerical Sort

  1. What it does: Sorts by numerical value
  2. When to use: With numeric data instead of text
  3. Example: 
    sort -n numbers.txt
5
10
20
100
Output will appear here
Original File: numbers.txt
