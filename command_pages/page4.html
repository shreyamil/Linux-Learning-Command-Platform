Working With Files: cat

File Content Commands

The cat command is used to display and manipulate file contents:

Command Description
cat file Display contents of a file
cat -b Add line numbers to non-blank lines
cat -n Add line numbers to all lines
cat -s Squeeze blank lines into one line
cat -E Show $ at end of each line
cat > file Create new file with content
cat >> file Append content to existing file

cat file - Display File Contents

  1. What it does: Shows the contents of one or more text files
  2. When to use: To quickly view file contents
  3. Example: 
    cat example.txt
Hello World
This is a sample file.

cat -b - Number Non-Blank Lines

  1. What it does: Adds line numbers to non-empty lines
  2. When to use: When you need to reference specific content lines
  3. Example: 
    cat -b example.txt
     1  Hello World
     2  This is a sample file.

cat -n - Number All Lines

  1. What it does: Numbers all lines including empty ones
  2. When to use: When you need absolute line numbers
  3. Example: 
    cat -n example.txt
     1  Hello World
     2
     3  This is a sample file.

cat -s - Squeeze Blank Lines

  1. What it does: Reduces multiple blank lines to single ones
  2. When to use: To clean up files with excessive spacing
  3. Example: 
    cat -s spaced.txt
Hello World

This is a sample file.

cat -E - Show Line Ends

  1. What it does: Displays $ at the end of each line
  2. When to use: To identify line endings and trailing spaces
  3. Example: 
    cat -E example.txt
Hello World$
This is a sample file.$

cat > file - Create File

  1. What it does: Creates a new file with content from terminal
  2. When to use: To quickly create small text files
  3. Example: 
    cat > newfile.txt
Type your content here
Press Ctrl+D to save

Created file contents:

cat >> file - Append to File

  1. What it does: Adds content to the end of an existing file
  2. When to use: To update files without overwriting
  3. Example: 
    cat >> existing.txt
Additional content
Press Ctrl+D to append
