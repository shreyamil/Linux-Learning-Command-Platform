Working With Directories: mkdir

Directory Creation Commands

The mkdir command creates new directories:

Command Description
mkdir dirname Basic directory creation
mkdir -p parent/child Creates parent and child directories
mkdir --parents parent/child Same as -p (long form)
mkdir -p parent/{child1,child2} Creates multiple subdirectories

mkdir dirname - Basic Directory Creation

  1. What it does: Creates a single directory
  2. When to use: When you need a new empty directory
  3. Example: 
    mkdir projects
# Creates a 'projects' directory

mkdir -p path/to/dir - Parent Directories

  1. What it does: Creates all directories in path
  2. When to use: When parent directories don't exist
  3. Example: 
    mkdir -p project/source/code
# Creates project/, project/source/, and project/source/code/

mkdir -p parent/{dir1,dir2} - Multiple Directories

  1. What it does: Creates multiple subdirectories
  2. When to use: When you need several directories at once
  3. Example: 
    mkdir -p website/{images,styles,scripts}
# Creates website/images/, website/styles/, and website/scripts/
